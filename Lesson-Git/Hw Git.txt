Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?


Как получить на локальный компьютер изменения с github.com

Какую информацию нам дают команда :
git status
git log
git remote -v 

1 чтобы скопировать репозиторий из гитхаб , предварительно нам нужно добавить в свой акк 
в гитхаб ssh Key ключь из гитбаша
2 потом чтобы склонировать репозиторий нажимаем <>code на гитхабе и выбираем 
протокол ssh и нажимаем copy и в гитбаше возрашаемся в тот репозиторий куда нам нужно
скачать потом набираем команду git clone и вставим ссылку из гитхаба

команда git status - покажет нам состояние файлов в репозитории гит 
Untracked files и git add <file> и имя файла

git log - нам показывает всех коммитов которые было сделано в репозитории 
которые были подписаны авторами
(Время и Дата и уникальные хешкоды для коммита)


с помошью  git remote -v мы убедимся что мы привязаны к гитхабу и оно дает нам 
origin push - отправлять в гитхаб
origin fetch - получать их гитхаб
